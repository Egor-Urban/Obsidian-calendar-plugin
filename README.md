# Obsidian-calendar-plugin
Edited version of popular plugin: https://github.com/liamcain/obsidian-calendar-plugin

I just added the ability to change the note creation location by day. There might already be similar forks, but I don't care :) 

Maybe i will update it. If i will need it. 


Just copy calendar folder to {OBSIDIAN VAULD}/.obsidian/plugins and restart Obsidian. New setting will be in calendar plugin settings
